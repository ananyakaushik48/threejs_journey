# Chapter 3

#### 1. Setting up a Vite Project
>
> * Create the directory and go into that directory
> ```bash
>$ mkdir first_three && cd first_three
>```
> * Initialise the npm project
> ```bash
>$ npm init -y
>```
> * Install Vite into the project
> ```bash
>$ npm i vite
>```
> * Update the ```package.json```
> ```json
>{
  Hello: "djfidjf"
}
>```
